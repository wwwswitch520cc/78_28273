– 四个独特的怪物可玩角色：Megadon、Ganira、Aegis Prime 和 Tempest Galahad<br>

– 基于真实世界位置的完全可破坏的城市<br>
-残酷的狂暴技能、毁灭性的灾变攻击和无情的终结技<br>
-凶猛的老板和数十种不同的敌人类型<br>
– 使用 DNA 增强来定制你的怪物</p>
